## [cleanup.js](cleanup.js)
cleanup.js — это скрипт для чистки таблицы manga_voiceovers в data.json от мусорных записей, которые автопарсер
  ошибочно прицепил к тайтлам манги.

  Что именно удаляет:
                                                                                                                          1. По ключевым словам (rejectRe) — реакции, обзоры, трейлеры, AMV, опенинги, мемы, клипы, топы, дорамы/фильмы,
  прохождения игр, Minecraft/Roblox/Gacha, шортсы и т.д.                                                                  2. Короткие названия (1 слово, <15 символов) — например «Некромант», «Реальность», «Стальной» — требует, чтобы
  название было либо в кавычках, либо сразу после «озвучка манги», либо в короткой части заголовка рядом с
  «озвучка/манга/главы». Иначе слишком велик риск, что слово случайно совпало.
  3. Двусложные названия — требует контекст «озвучка/манга/главы/диапазон глав» в той же части.
  4. Совпадение слов — 3+ значимых слов должны совпасть (минимум 60% для длинных, все — для коротких), русские
  стоп-слова игнорируются.

  Всё что не прошло — пишется в лог [KEYWORD]/[1WORD]/[2WORD]/[FEWWORDS]/[NOMATCH] и удаляется. Результат сохраняется
  обратно в data.json. Запускается вручную: node cleanup.js.
